# manaba scraping tool

## install

依存するライブラリは

- selenium
- beautifulsoup4
- lxml

### using conda

```:shell
conda install selenium beautifulsoup4 lxml
```

### pip

```:shell
pip install selenium beautifulsoup4 lxml
```

### ID and password

`scraping.py`と同じフォルダに`secret.txt`を作成し、1行目にID、2行目にパスワードを書く
